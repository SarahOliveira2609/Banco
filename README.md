# 🏦 OliverBank — Sistema Bancário em Python

Sistema bancário simples de terminal desenvolvido em Python como projeto prático durante meu aprendizado de programação.

---

## 📋 Sobre o projeto

O OliverBank simula as operações básicas de um banco pelo terminal. O usuário pode se cadastrar, fazer login com validação de senha e realizar operações financeiras com histórico registrado.

---

## ⚙️ Funcionalidades

- ✅ Cadastro de cliente (nome, CPF, telefone, e-mail, senha)
- ✅ Login com autenticação e limite de tentativas
- ✅ Consulta de saldo
- ✅ Depósito
- ✅ Saque (com validação de saldo)
- ✅ Transferência (com validação de saldo)
- ✅ Histórico de transações

---

## 🛠️ Tecnologias utilizadas

- Python 3
- Biblioteca `getpass` (entrada de senha segura)
- Biblioteca `time` (delays para melhor UX no terminal)
